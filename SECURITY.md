# Security policy

## Reporting a vulnerability

If you discover a security vulnerability, **do not** open a public issue. Please report it in one of the following ways:

- **Email**: send to the maintainer address (see README or GitHub profile).
- **GitHub Security Advisories**: if enabled for the repo, use the [Security tab](https://github.com/your-username/landing/security) → “Report a vulnerability”.

We will respond as soon as possible and will update you when the issue is addressed or when we need more information.

## Scope

- This library mainly handles JSON config and UI rendering; it does not store passwords or sensitive data.
- Issues in dependencies (e.g. Zod, React, Next.js) should be reported to the respective projects when appropriate; you may still notify us so we can update dependencies or document the issue.

Thank you for helping keep the community safe.
